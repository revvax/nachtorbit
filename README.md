# ACTS. LOVE.

Persönlicher Festival-Planer mit Hörproben, Sterne-Wertung und Timetable — als installierbare Web-App, offline nutzbar.

In ~4 Stunden in Hamburg vibe-coded von **Malvin Bonnin**.

**Live:** https://revvax.github.io/acts-love/app/

## Features
- Hörproben: SoundCloud & YouTube spielen inline
- Sterne-Wertung (1–5) pro Act — lokal & privat auf dem Gerät, offline
- Timetable nach Stage × Uhrzeit, Filter nach Tag / Genre / Stage
- A–Z-Schnellscroll, Pull-to-Refresh
- Installierbar („Zum Home-Bildschirm"), läuft offline (Service Worker)
